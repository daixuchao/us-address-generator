Canada Address Format Guide

Updated July 13, 2026

Canadian addresses look similar to U.S. addresses, but the province and postal code rules are different. Test forms should not assume every North American address uses a U.S. state and ZIP Code.

Common layout

Alex Martin
240 Queen St W, Unit 1204
Toronto ON M5V 1Z7
Canada

Fields to support

  • Recipient: person or company name.
  • Street line: civic number, street name, and optional unit.
  • Municipality: city or town name.
  • Province: two-letter code such as ON, QC, BC, AB, or NS.
  • Postal code: alternating letters and numbers, usually written as A1A 1A1.

QA notes

Include postal codes with spaces, province abbreviations, long municipality names, bilingual street names, and unit fields. Do not validate Canadian records with U.S.-only ZIP Code rules.
